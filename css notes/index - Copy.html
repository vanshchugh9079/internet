CSS

defenation

CSS is a style sheet language used for describing the look and formatting of a document written in a markup language like HTML. It enables developers to control the layout, colors, fonts, spacing, and other visual aspects of web pages.

how does css work

CSS works by targeting HTML elements and applying styles to them. To apply styles, you use selectors to target specific HTML elements these selector are as tagname,class,id or other attributes, and then define the appearance properties for those elements.

css syntax :

selector {
        property: value;
        /* more properties and values */
    }

types of stylesheet

  • inline

    Inline styles are applied directly to individual HTML elements using the style attribute.

    <"p style="color: blue; font-size: 16px;">This is a paragraph with inline styles.

  • internal

    Internal styles are defined within the <"style> tag in the <"head> section of an HTML document.

    syntax
        <"head>
            <"style>
                p {
                    color: blue;
                    font-size: 16px;
                }
            
     <"/head>

  • external

    External stylesheets are separate files with a .css extension that contain CSS rules and are linked to an HTML document using the <"link> tag.

    syntax
    <"link rel="stylesheet" href="styles.css">

define css propery

a CSS property refers to an attribute or characteristic that can be applied to HTML elements to define their appearance, layout, or behavior on a webpage

deine class and id

  • class

    a class is a selector that allows you to apply a style to one or multiple HTML elements. Classes are defined in CSS using a period (.) followed by the class name.

  • id

    an ID is a selector used to uniquely identify a single HTML element on a web page. IDs are defined in CSS using the pound sign (#) followed by the ID name.

use of id and classes in CSS

  • Id

    usage of id

    • IDs are meant to uniquely identify a single HTML element on a page. Each ID should be unique within the HTML document.

    • They are often used when you want to apply specific styles to a single element.

    • IDs are particularly useful for targeting a specific element for JavaScript manipulation

  • class

    usage of class

    • Classes are used to apply styles to multiple HTML elements that share the same class attribute.

    • Unlike IDs, classes can be applied to multiple elements on the same page

    • They're commonly used for styling groups of elements that have similar characteristics or need consistent styling.

background properties in CSS

  • background-color

    Sets the background color of an element. You can use color names, hexadecimal values, RGB, or RGBA values

  • background-image

    Specifies an image to be used as the background. You can use url() to link to an image file.

  • background-repeat

    Defines how a background image should be repeated, such as repeat, repeat-x, repeat-y, or no-repeat.

  • background-position

    Sets the initial position of a background image. Values like center, top, bottom, left, right, or specific lengths/percentages can be used.

  • background-size

    Specifies the size of the background image. You can set it to cover, contain, specific lengths, or percentages.

  • background-attachment

    Determines whether the background image scrolls with the content. Values include scroll, fixed, or local.

you can also do all properties in a single line by folloeing shorthand properties
.example {
        background: #f0f0f0 url('example.jpg') no-repeat center top / cover;
      }

border properties in CSS

In CSS, the border property and its related properties are used to create borders around elements. Borders can be applied to various HTML elements like divs, paragraphs, buttons, etc., and they can be styled, colored, and sized according to your design needs.

  • border-width

    Sets the width of the border. You can specify it in pixels, ems, rems, percentages, or other length units.

  • bordr-style

    Specifies the style of the border. Values can be solid, dashed, dotted, double, groove, ridge, inset, outset, or none.

  • border-color

    Sets the color of the border. You can use color names, hexadecimal values, RGB, or RGBA values.

  • border-radius

    Applies rounded corners to the border. You can use percentages or lengths to specify the radius.

you can also do all border properties in a single line by folloeing shorthand properties
        .example {
            border: 2px dashed #f00;
            border-radius: 10px;
          }

Box model

The box model is a fundamental concept in CSS (Cascading Style Sheets) that defines how elements on a webpage are structured in terms of their dimensions and spacing. It consists of four essential components: content, padding, border, and margin.

  • content

    This refers to the actual content of the HTML element, such as text, images, videos, etc.

  • padding

    It is the space between the content and the element's border. Padding can be set using properties like padding-top, padding-right, padding-bottom, and padding-left.

  • border

    This is the boundary around the padding. It can have properties like border-width, border-style, and border-color to control its appearance.

  • margin

    Margin is the space outside the element's border. It creates a gap between the element and other elements around it. Properties like margin-top, margin-right, margin-bottom, and margin-left are used to define margins.

Content

Margin

In web design and CSS (Cascading Style Sheets), the "margin" refers to the space around an HTML element, outside its border, separating it from other elements. It creates an outer space or gap between the element and its neighboring elements.


The margin property in CSS allows you to control the space outside the border of an element. It can be set for individual sides (top, right, bottom, left) or all sides at once.


padding

In web design and CSS (Cascading Style Sheets), "padding" refers to the space between the content of an HTML element and its border. It provides internal spacing within the element, separating the content from the border.


The padding property in CSS allows you to control the space between an element's content and its border. Padding can be set for individual sides (top, right, bottom, left) or for all sides at once.
